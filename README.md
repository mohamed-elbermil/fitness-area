# Fitness-Area 💪🏼
Fitness-Area est un projet initialement développé lors d’un bootcamp en HTML, CSS et JavaScript. Au fil du temps, je l’ai continuellement amélioré, tant sur le plan esthétique que fonctionnel. Récemment, le projet a été migré vers React afin d’optimiser sa performance et sa maintenabilité.

🛠 Stack

- React

- NodeJs

- CSS

# Installation

- git clone https://github.com/mohamed-elbermil/fitness-area.git
- cd movie fitness-area
- npm i

# lancement du projet

- npm run dev
- Choisir le port 3000
